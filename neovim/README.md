# Neovim
    ├── .config
    │   └── nvim
    │       └── init.vim
    └── README.md

## Requirements
- `neovim` <<https://github.com/neovim/neovim>>
- `vim-plug` <<https://github.com/junegunn/vim-plug>>

## Installation
You can install this package with

    $ rm -f ~/.config/neovim/init.vim
    $ stow neovim
